# 第六单元 — CSS 练习（3/18）
CSS 练习：编写规则与语义化代码。
 
---
 
## 任务一 — 添加缺失的语义化标签
 
你的初始代码缺少语义化 HTML 标签。请添加以下标签：
 
- `<header>`
- `<main>`
- `<footer>`
- `<head>`
- `<body>`
 
挑战在于：**判断哪些 HTML 元素（子元素）应该放入哪个语义化标签（父元素）中**。
 
---
 
## 任务二 — 让你的代码样式与黑板上的一致
 
使用以下 CSS 属性完成样式设置：
 
- `color`（颜色）
- `text-align`（文本对齐）
- `text-shadow`（文字阴影）
- `background-color`（背景颜色）
- `text-decoration-line`（文本装饰线）
- `text-decoration-style`（文本装饰样式）
- `background-image`（背景图片）
 
你需要总共编写 **6 条 CSS 规则**。挑战在于搞清楚：
1. 哪些 HTML 标签需要添加规则
2. 每条规则里应该包含哪些属性
