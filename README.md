# Unit 6 — CSS Practice (3/18)
CSS practice: Writing rules and making code semantic.
 
---
 
## Task 1 — Add the Missing Semantic Tags
 
Your starter code is missing its semantic HTML tags. Add the following:
 
- `<header>`
- `<main>`
- `<footer>`
- `<head>`
- `<body>`
 
The challenge is figuring out **which HTML elements (children) belong inside which semantic tag (parent)**.
 
---
 
## Task 2 — Style Your Code to Match the Board
 
Use the following CSS properties to complete the styling:
 
- `color`
- `text-align`
- `text-shadow`
- `background-color`
- `text-decoration-line`
- `text-decoration-style`
- `background-image`
 
You will need to write **6 CSS rules** in total. The challenge is figuring out:
1. Which HTML tags need a rule
2. Which properties belong inside each rule
 
Good luck! 🎯
